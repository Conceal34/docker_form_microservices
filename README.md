# Frontend (Node.js with Express):

Created a frontend using Express and Node.js.
Included a form
Configured the form to send a request to the Flask backend.


# Backend (Flask):
Used the Flask backend to handle the form submission and process the data.

# Folder Structure:
Organized the project with separate folders for the frontend and backend microservices.

# Docker Configuration:
Created the Docker files for both the frontend and backend.
wrote a .yaml file (Docker Compose) to connect both services in the same network.
